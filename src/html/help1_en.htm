What is a syntax diagram?

The syntax diagram graphically represents the structure of the syntactic unit.

Each syntax diagram has a name that uses the corresponding meta-variable as its

The syntax diagram is an oriented graph with marked edges. Meta-variables and meta-constants are used to mark the edges. Metacharacters on the syntax diagram are not used
Therefore, meta-constants in the syntax diagrams are not enclosed in quotes.

Meta-variables are enclosed in angle brackets < >.

To separate the name of the syntax diagram from the graph, use the metacharacter ::=

The symbols < > and ::= are the only metacharacters used.

The meta-variable on the marked edge of the graph means that the corresponding fragment of the diagram must be detailed by substituting the syntactic diagram with the name corresponding to this meta-variable.
