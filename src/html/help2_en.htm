Syntax Diagrams is a program for drawing syntax diagrams. What are the syntax diagrams, you can find out in the corresponding menu item.

The selection of shapes for drawing is done in ToolBar:

There are 5 main figures in the program:

  • - a figure corresponding to the header of the syntax diagram.
  • - a figure corresponding to the "Meta-variable".
  • - a figure corresponding to the "Metaconstant".
  • - a figure corresponding to the line
Also there is a mode of moving / editing shapes without drawing -

Start drawing

If the "line" mode is selected, each click LMB on the canvas adds a new point to the shape. Clicking RMB - stops drawing the current shape. Otherwise, if you select one of the modes for drawing text figures - each click adds a new shape.

As text for figures (except lines), the value of the field "Text in ToolBar":

To edit shapes - you must point to the desired point (To move - to the center of the figure, to move the side - to the side, to move the vertex - to the top), then, while holding the left mouse button, move the cursor. To change the text inside the shape - you need to click on the shape and change the value of the "Text" field.

To change the size of the canvas - you need to go to the appropriate section of the menu (Settings - Canvas size).

The rules of "drawing" shapes by the program

To simplify the life of users, the program performs the following manipulations:

  • If the lines are placed close, they are "magnetized" to each other
  • If the line is vertically, an inclined line is added to its beginning / end (see examples)
  • An arrow is added to the center of the penultimate "substring."
  • An arrow is added to the end of the line
  • If the line is horizontal and leaves a different line, an inclined line is added to the beginning
  • If the line is horizontal and enters another, an inclined line is added to the end
  • and others
Examples can be found by opening the folder "examples"
